## Cassie戦を理解する

Cassieはプレイヤーにとって大きな実力チェックとなるボスです。Demo 5でも彼女の行動パターン自体は一貫していますが、高難易度モディファイアとボスラッシュモードの導入により、許されるミスは少なくなりました。彼女は突進系の物理攻撃と飛び道具を組み合わせてきて、プレイヤーには「Perfect Guard」メカニクスの習得が求められます。

| 攻撃名 | 説明 | 対策 |
| :--- | :--- | :--- |
| **Lunge Poke** | レイピアで素早く前方に突進する。 | 白く光った瞬間に合わせてガード。 |
| **Rose Volley** | 複数の飛び道具を弧状に投げる。 | ガードボタンを素早く連打して各ローズをパリィ。 |
| **Thorn Trap** | 地面からツタを召喚する。 | 床のエフェクトを見たらジャンプして回避。 |
| **En Garde** | 物理攻撃をカウンターする防御姿勢。 | スリングショットのようなアイテム・遠距離攻撃を使う。 |

> **警告:** Cassieが「En Garde」姿勢中に近接攻撃をすると、即座に大ダメージのカウンターを受けます。接近する前に、必ず姿勢を解除するのを待ちましょう。

## Demo 5向けスリングショットビルド

Demo 5メタで特に興味深い変化のひとつが、スリングショットビルドの台頭です。多くのプレイヤーはグレートソードの火力やダガーの速度を好みますが、スリングショットはCassieの近距離カウンターの多くを安全に回避できる、独自の遠距離アドバンテージを持っています。

最近のコミュニティ検証によると、スリングショットには意識すべき明確なダメージ上限があります。通常、1ヒットあたりの最大ダメージは8で頭打ちになります。これは高性能な近接武器より低く見えるかもしれませんが、距離を取って攻撃でき、特定のボス反応を回避できる点から、**block tales demo 5 cassie** 戦では非常に有力です。

### 「Ants Up」が危険な理由
スリングショットビルドを組む際、多くのプレイヤーはダメージ強化のために「Ants Up」バッジを付けたくなります。しかしDemo 5では、とくにスリングショットに関してはこれがミスになりがちです。「Ants Up」バッジは、スリングショットのアクションコマンドにある緑のタイミングバーを大幅に縮小します。結果としてタイミング猶予は非常に狭くなり――薄いパンの切れ端のようなレベル――高圧なボス戦で安定して「Great」「Excellent」を出すのがほぼ不可能になります。

## 推奨バッジ構成

Cassie戦での効率を最大化するには、バッジ構成をBP（Badge Points）管理と防御ユーティリティ重視にするべきです。**block tales demo 5 cassie** 戦は消耗戦になりやすいため、ダメージを出すことと同じくらい、体力維持が重要です。

| バッジ名 | コスト (BP) | 効果 | 優先度 |
| :--- | :--- | :--- | :--- |
| **Power Plus** | 3 | 基礎ダメージを1上昇。 | **高** |
| **Defend Plus** | 3 | 基礎防御力を1上昇。 | **高** |
| **Quick Change** | 2 | ターンを消費せずにアイテムを切り替え。 | **中** |
| **Happy Flower** | 2 | ターン中にランダムでSPを回復。 | **低** |
| **Pretty Lucky** | 2 | 攻撃を回避させる小確率が発生。 | **中** |

これらのバッジをスリングショットと併用することで、堅い防御を保ちながらCassieのHPを着実に削れます。もしスリングショットのタイミングが難しすぎるなら、「Power Plus」を「HP Plus」に替えて生存力を上げるのも有効です。

## Cassie攻略ステップ

以下の手順を実行すれば、複数フェーズの戦闘を効果的に進められます。

1.  **開幕の一手**: 戦闘開始時に、自己強化アイテムまたはCassieへのデバフを使用します。スリングショットを使うなら、序盤は「Excellent」を狙ってSPを溜めることを重視しましょう。
2.  **取り巻きの処理**: Cassieは注意をそらすために小型の敵を召喚することがあります。無視は禁物です。Fire Flowerのような複数対象スキル・アイテムで一掃し、再びボスに集中しましょう。
3.  **中盤（HP50%）**: CassieのHPが50%になると速度が上がります。ここでスリングショットビルドが真価を発揮します。距離を保ち、速くなったLunge Pokeに対するガードタイミングに集中できます。
4.  **終盤戦**: 体力バーが赤くなると、「Thorn Trap」の使用頻度が上がります。飛び道具の被弾で倒されないよう、常にHPを最低10以上維持してください。

> **💡 ヒント:** 「Cheeseburger」または同等の回復アイテムを常に2個以上持っておきましょう。Demo 5版のこの戦闘は、長い第2フェーズを耐え切れるプレイヤーほど有利です。

## 応用戦術：スリングショットを極める

スリングショット運用に特化するなら、アクションコマンド習熟が2ダメージと8ダメージを分けます。1回のタイミング入力で済む剣と異なり、スリングショットは「引いて離す」メカニクスです。 

*   **離すタイミング**: 緑バーは高速で動きます。カーソルが緑ゾーン中央に入った瞬間にボタンを離しましょう。
*   **距離要素**: スリングショットは遠距離武器ですが、Block Talesでは敵との距離がダメージを厳密に変えるわけではありません。ただし弾道の見え方は変わるため、集中力には影響します。
*   **SPシナジー**: スリングショットで得たSPを防御スキルに回しましょう。「Ants Up」を使わないためタイミング猶予が広く、ほぼ毎ターン安定してSPを獲得できます。

## FAQ

**Q: スリングショットビルドはDemo 5の全コンテンツで通用しますか？**

A: スリングショットビルドは **block tales demo 5 cassie** 戦や、カウンター姿勢を持つ他ボスには非常に有効です。ただし、デモ後半に出る高防御の「タンク」系敵には苦戦しやすいです。特定ボス向けの特化ツールとして使うのが最適です。

**Q: なぜスリングショットで「Ants Up」を避けろと言われるのですか？**

A: 「Ants Up」バッジはダメージを上げる一方で、「Action Command」の成功猶予を大きく悪化させます。スリングショットでは成功ゾーンが極端に小さくなるため、多くのプレイヤーがタイミングを外し、結果としてバッジなしの方が総ダメージが高くなることが多いからです。

**Q: Block Talesでスリングショットはどうやって解放しますか？**

A: スリングショットはメインハブエリアの武器ショップで入手するか、序盤チャプター内の宝箱から見つけられます。Demo 5でCassieに挑む前に、必ず最大レベルまで強化しておきましょう。

**Q: スリングショットでCassieに与えられる最大ダメージは？**

A: 現行バージョンでは、完璧なタイミングのスリングショットは通常8ダメージが上限です。ただし「Power Plus」のようなバッジでわずかに変動する可能性があります。それでも重要なのは最大値より、ヒットの安定性です。
